About

CPU Charity allows a user to donate mined crypto to a good cause just by leaving this webpage open. The web application asks the user if it could borrow some of the processing power so in the background it could complete hash sets. We've limited the activity to not impede normal daily tasks. Using the combined efforts of many people it increases the mining capabilities resulting in a bigger donation to charity.


This is a development project and just a proof of concept.
